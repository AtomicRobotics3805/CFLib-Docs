# With Groovy

Install CFLib using the `Groovy` syntax

<details>
<summary>Step 1: Add the repositories.</summary>

In your root `build.gradle` file (`your-project/build.gradle`), find the `allprojects/repositories` block. It should look like this:

```groovy
allprojects {
    repositories {
        mavenCentral()
        google()
    }
}
```

Add these lines to the end of the block:

```groovy
maven { url = 'https://jitpack.io/' }
maven { url = 'https://maven.brott.dev' }
```

It should now look like this:

```groovy
allprojects {
    repositories {
        mavenCentral()
        google()

        maven { url = 'https://jitpack.io/' }
        maven { url = 'https://maven.brott.dev' }
    }
}
```
</details>

<details>
<summary>Step 2: Add the dependencies.</summary>

In your `TeamCode/build.gradle` file, find the `dependencies` block. It should look something like this:

:::caution

Make sure you are in the *project* `build.gradle`, not the root `build.gradle`

:::

```groovy
dependencies {
    implementation project(':FtcRobotController')
    annotationProcessor files('lib/OpModeAnnotationProcessor.jar')
}
```

Add these lines at the end of the block

```groovy
implementation "com.github.AtomicRobotics3805:CFLib:1.0.0"
implementation 'org.apache.commons:commons-math3:3.6.1'
implementation 'com.acmerobotics.roadrunner:core:0.5.5'
implementation 'com.github.NoahBres:MeepMeep:2.0.1'
```

So it looks like this

```groovy
dependencies {
    implementation project(':FtcRobotController')
    annotationProcessor files('lib/OpModeAnnotationProcessor.jar')

    implementation "com.github.AtomicRobotics3805:CFLib:0.2.0"
    implementation 'org.apache.commons:commons-math3:3.6.1'
    implementation 'com.acmerobotics.roadrunner:core:0.5.5'
    implementation 'com.github.NoahBres:MeepMeep:2.0.1'
}
```

</details>

Finally, sync Gradle.
