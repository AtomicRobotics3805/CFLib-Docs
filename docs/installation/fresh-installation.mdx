# Fresh installation with the FTC SDK

CFLib can be downloaded along with the [FTC SDK](https://github.com/FIRST-Tech-Challenge/FtcRobotController) to minimize the time spent installing it. 

## Clone the Git repository
Assuming you have Git installed (if you don't, go to [https://git-scm.com/book/en/v2/Getting-Started-Installing-Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) and follow the instructions), enter the following command:
```bash
git clone https://github.com/AtomicRobotics3805/CFLib-quickstart.git
```

## Download the latest release

[Click to download](https://github.com/AtomicRobotics3805/CFLib-quickstart/archive/refs/tags/0.1.0.zip)

## GitHub CLI

```bash
gh repo clone AtomicRobotics3805/CFLib-quickstart
```
